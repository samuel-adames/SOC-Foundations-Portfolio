# SOC Foundations Portfolio

This repository documents my learning path and hands-on practice related to Security Operations Center (SOC) fundamentals.

The objective of this project is to demonstrate my understanding of how a SOC operates, how alerts are analyzed, and how incidents are triaged and classified, focusing on a SOC Tier 1 / Junior Analyst role.

## Learning Objectives
- Build a solid foundation of SOC concepts
- Understand detection and response processes
- Learn the role of people, processes, and technology in a SOC
- Perform practical alert analysis exercises

## Topics Covered
- SOC structure and responsibilities
- SOC roles and tiers
- Incident detection and response
- Alert triage and prioritization
- SIEM, EDR, and firewall fundamentals
- Practical analysis of a network port scan alert

## Practical Experience
This repository includes a hands-on alert triage exercise where a potential port scan was analyzed, investigated, and classified as a false positive after contextual validation.

## Tools & Concepts
- SIEM
- EDR
- Firewalls
- Alert triage
- 5W analysis method

## Disclaimer
All scenarios are part of a learning environment and do not represent real production systems.
